### Estilos de texto
Aquí se muestran las diferentes formas de escribir texto dentro de archivos .md (Markdown)
- Texto normal
* *Cursiva* _Cursiva_
- **Negrita** __Negrita__
* ~~Tachado~~
- ***Mix little mix***
* <sub> Texto bajo </sub> <sup> Texto alto </sup>
- <ins> Textosubrayado </ins>
* > Entrecomillado
